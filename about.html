About Weather App

The Weather App is a sleek and responsive web application that gives users real-time weather updates based on their current location or a city they enter. Designed with a visually rich interface using Bootstrap 5 and modern APIs, it offers:

  • Location-based weather updates
  • Real-time temperature, humidity, and condition
  • Clean and responsive design using Bootstrap 5
  • Powered by Open-Meteo and IPAPI APIs

This app was built as part of a project to combine usability with visual design, and is ideal for both desktop and mobile users.
